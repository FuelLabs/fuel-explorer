## Configuration

The faucet makes use of environment variables for configuration.

| Environment Variable | Description                                      |
| -------------------- | ------------------------------------------------ |
| FUEL_PROVIDER_URL    | Fuel node provider URL.                          |
| PORT                 | Define what port the explorer should be bind to. |
